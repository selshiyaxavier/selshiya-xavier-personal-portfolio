# VJ Selshiya Portfolio

A premium "Cyber-chic" portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Setup Instructions

1.  **Navigate to the project directory:**
    ```bash
    cd vj-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the website:**
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

*   **Cyber-chic Design:** Dark theme with neon pink/purple accents.
*   **Responsive:** Fully optimized for mobile and desktop.
*   **Animations:** Smooth scrolling, glassmorphism, and Framer Motion effects.
*   **Sections:** Hero, About & Stats, Experience Timeline, Portfolio Grid, Contact.

## Tech Stack

*   Next.js 14 (App Router)
*   Tailwind CSS
*   Framer Motion
*   Lucide React Icons
*   Montserrat Font
